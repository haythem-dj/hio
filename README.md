# HIO (Haythem IO)

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

A self hostable application hub

## Features

- Fully self hostable with docker container
- Multiple self hostable apps in one place

## Installation

```bash
git clone https://github.com/haythem-dj/hio.git
cd hio
./configure
```
